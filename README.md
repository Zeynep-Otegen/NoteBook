# NoteBook APP
Basit bir not defteri uygulamasıdır. Bu uygulama, kullanıcıların kolayca not eklemesini, kaydetmesini ve silmesini sağlar.

Zip dosyasındaki sadece Notebook yazan uygulamaya tıklayın.

İndir:

# Özellikler
Not Ekleme: Yeni notlar oluşturma imkanı.

Not Kaydetme: Oluşturulan notları kalıcı olarak kaydetme.

Not Silme: Artık ihtiyaç duyulmayan notları listeden kaldırma.

Arama Çubuğu (Geliştirme Aşamasında): Notlar arasında arama yapmak için bir arama çubuğu mevcuttur. Ancak bu aşamada arama işlevi henüz aktif değildir ve geliştirilme beklemektedir.
